Le projet en quelques mots

Le projet Rugby Tropical Cup est un site internet complété par deux pages web mobile. La plateforme recense les matchs de rugby et les informations concernant l'évenements en lui même, les utilisateur avec un ticket ont la possibilité de les scanner pour accéder rapidement aux détails spécifiques de chaque match, comme les équipes, les dates et le stade.

Usage de Vues

Les vues API définissent comment les informations sont préparées et envoyées en réponse aux demandes. Elles s'occupent de rendre les données faciles à comprendre pour les applications qui utilisent l'API.

Données Json

le JSON ressemble à une liste de choses, chaque chose ayant un nom et une valeur associée. C'est un moyen efficace pour les applications de partager des données de manière lisible et structurée.

Base de Données

une base de données permet de ranger et d'organiser les données, puis de les récupérer rapidement quand on en a besoin. On utilise un langage de requête (comme le SQL) pour interagir avec la base de données et effectuer des opérations telles que la création, la lecture, la mise à jour et la suppression de données.

L'appel API

un appel API est comme une conversation entre deux applications, où l'une demande quelque chose et l'autre répond avec les informations demandées.

la page Mobile

Voici la page Mobile, créée en Mobile First, cette page permet de retrouver tout les matchs, où l'on retrouve le stade où aura lieu l'affrontement, avec les équipes locals et visiteurs, et enfin la date et l'heure

Le Scanneur

le Scanneur, est une application web mobile, qui permet de scanner des QR codes présent sur les billets pour aller voir les matchs, cela affiche, la catégorie, le prix du billet, la place, et de quel match il s'agit,
